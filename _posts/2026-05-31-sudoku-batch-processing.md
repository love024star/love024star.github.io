# AI Agent 批量处理数独题库

日期：2026-05-31  

## 任务背景


手头有一批数独题库素材，来自训练App截图，需要按统一排版输出为可打印的练习文档。


9种题型 ：九宫标准、六宫标准、六宫堡垒、六宫不规则、六宫奇偶、六宫对角线 、六宫无马 、六宫连续 、四宫杀手  



目标产出：每个题型生成一个Word文档 ，A4排版 ，每页6题，（规则放入页眉 ）。



##技术方案 



|环节 ||技术选型 ||说明 |
|--- |--- |--- |
|规则提取||视觉大模型OCR||从首图提取规则文字|
|图片处理 ||Python + PIL||调整尺寸设置DPI|
|文档生成 。|python-docx|||页眉排版+表格布局 |

##踩坑记录  

### DPI缺失导致渲染异常  


原图缺少DPI元数据 ，Word用默认96DPI渲染 ，图片显示尺寸远超预期 。解决方法 ：插入前通过PIL强制设DPI为200。

```python  
img.info['dpi'] = (200,200)
```

### 图片尺寸调试 


最终版图片宽度设为6 .cm ,恰好一页6题 。

###页眉方案的妙处  


将标题和规则从正文移到页眉 ，释放正文空间给内容本身 。

##总结



整个任务通过AI Agent调度完成 ，涉及文件扫描 、图片处理、、OCR、、文档生成等多个环节 。迭代过程中保持上下文连续——这是对话式协作的优势 。
