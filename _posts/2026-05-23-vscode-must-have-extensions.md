VSCode 本身已经很强大了，但装对了插件可以让它变得更强。下面是我每天都在用的插件清单。

## 通用效率类

### 1. GitLens — Git 超级增强
一眼看到每一行代码的作者、历史、变更原因。Code Review 神器。

### 2. Todo Tree
把所有 TODO、FIXME、HACK 标记聚合到一个侧边栏，再也不会遗漏任务。

### 3. Error Lens
直接把错误和警告显示在代码行上，不用再切到底部面板去看。

## 代码编写类

### 4. Auto Rename Tag
改 HTML/JSX 开始标签，自动帮你改结束标签。

### 5. Thunder Client
在 VSCode 里直接发 HTTP 请求，不用再开 Postman。

### 6. GitHub Copilot
AI 辅助编程，写注释生成代码，补全整段函数。

## Markdown 写作类

### 7. Markdown All in One
TOC 生成、快捷键、表格格式化，写 Markdown 必备。

### 8. Markdown Preview Enhanced
支持导出 PDF、HTML，图表渲染，MathJax 数学公式。

## 推荐插件清单

1. GitLens
2. Todo Tree
3. Error Lens
4. Auto Rename Tag
5. Thunder Client
6. Markdown All in One
7. Markdown Preview Enhanced
8. Prettier - Code formatter

好的工具要经常用才能发挥价值，装完记得用起来！
